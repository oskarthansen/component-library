# Ripple

Import as `import Ripple from '@norlys/norlys-components/ripple'` or
`import { Ripple } from '@norlys/norlys-components/ripple'`

## Usage

Add the Ripple as child to the element you want to have the effect. <br/> This element should have
`position: relative`. <br /><br />

The ripple can trigger through its own event handler, but you can also pass an input `ClickEvent`
which will overwrite the default behavior. <br /> This input requires the following properties:

- `container`: The element being clicked (e.g. `event.currentTarget.getBoundingClientRect()`)
- `pageX`: X coordinate of click event (e.g. `event.pageX`)
- `pageY`: Y coordinate of click event (e.g. `event.pageY`)

These values can come from different Mouse events, like `onMouseDown`, `onClick`, etc.
