## Button customization

The button can be customized in multiple ways, depending on what you wish to do. The button has some
build-in functionality which can't and shouldn't be changed. The only thing that should be changed
is the colorscheme of the button. The easiest way to modify the colorscheme is through the <strong>Theme prop</strong>.

# Usage

The button component is as minimal as possible. The button has some build-in functionality which
can't be changed, but the colorpalette/colors of the button can.
