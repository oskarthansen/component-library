## Install

Komponentbiblioteket er skrevet i React. For at bruge komponentbiblioteket skal du installere en af
pakkerne; <strong>base-components</strong>, <strong>styled-components</strong>, osv..

#### base-components
<Source
  language="bash"
  code={dedent`
  npm install --save @norlys/base-components
  `}
/>

#### styled-components
<Source
  language="bash"
  code={dedent`
  npm install --save @norlys/styled-components
  `}
/>

#### themes
<Source
  language="bash"
  code={dedent`
  npm install --save @norlys/themes
  `}
/>

osv...


### Use

Import components you want into your UI

<code>{`import { Button, Typography } from '@norlys/styled-components';`}</code>

or

<code>{`import Button from '@norlys/styled-components/button';`}</code>
